## Node Type Descriptions

### Two Truths & a Lie (60 XP, ~3-4 min)
Three statements are presented. Player must identify which one is false. Includes explanation and learning points.

### Found Tape (40 XP, ~3-4 min)
Audio recording of a historical account (diary, testimony, letter). Player listens and answers comprehension questions.

### Headlines (60 XP, ~3-4 min)
Period newspaper front page with 3 headlines. Player reads and answers questions about the events.

### Quiz Mix (50 XP, ~2-3 min)
Multiple choice quiz covering chapter content. Quick knowledge check.

### Decision Point (60 XP, ~3-4 min)
Historical scenario where player makes a choice. Shows consequences and historical outcome.

### Boss Challenge (100 XP, ~5-6 min)
Comprehensive chapter test combining multiple question types. Must achieve high score to pass.

### Video Lesson (40 XP, ~3-4 min)
Watch video content with embedded questions or interactions.

### Image Explore (50 XP, ~3-4 min)
Interactive image/map with hotspots to discover information.

### Chrono Order (50 XP, ~3-4 min)
Drag events into correct chronological order.

### Video Trivia (varies, ~3-4 min)
Video plays with host asking question. Answers appear at trigger point. Response videos play based on correct/wrong answer.

## Admin Content Management

### Ghost Army Editor (`/admin/ghost-army`)
- Upload videos for each node (intro, narration, background)
- Set thumbnails and background images
- Configure node-specific media

### Trivia Editor (`/admin/trivia`)
- Create video-driven trivia sets
- Upload question videos
- Set answer trigger points (end of video or timestamp)
- Upload correct/wrong response videos
- Configure XP rewards per question

### Journey Editor (`/admin/journeys`)
- Edit arc, chapter, and node content
- Modify XP rewards
- Update text content

### Course Editor (`/admin/courses`)
- Manage learning courses
- Upload thumbnails
- Edit lessons and units

### Arcade Editor (`/admin/arcade`)
- Configure mini-games
- Upload game images
- Set difficulty and rewards
